# 代码编辑器优化说明

## 改进内容

### 1. 界面简化
- **移除了冗余的工具栏按钮**：Monaco Editor 本身支持查找（Ctrl+F）、替换（Ctrl+H）等功能，无需额外按钮
- **保留了必要的操作按钮**：编辑、保存、关闭等核心功能
- **优化了布局结构**：文件信息放在左侧，操作按钮放在右侧，更加清晰

### 2. 样式优化
- **统一了颜色方案**：使用 VS Code 风格的深色主题配色
- **改进了边框和背景**：使用实色而非半透明，避免"卡片嵌套"的视觉问题
- **优化了间距和对齐**：使界面更加整洁美观

### 3. Monaco Editor 配置优化
- **启用了内置查找/替换功能**：用户可以通过 Ctrl+F/Ctrl+H 使用
- **优化了上下文菜单**：右键菜单提供更多选项
- **启用了快速建议**：提供更好的代码补全体验
- **自动换行默认开启**：提升可读性

### 4. ✅ 中文本地化已实现
使用 `monaco-editor-nls` 包实现了完整的中文界面支持：
- 所有菜单、按钮、提示都显示为中文
- 查找/替换对话框中文化
- 右键菜单中文化
- 状态栏信息中文化

**技术实现**：
```javascript
import { setLocaleData } from 'monaco-editor-nls'
import zh_CN from 'monaco-editor-nls/locale/zh-hans'

// 在创建编辑器之前调用
setLocaleData(zh_CN)
```

## 快捷键说明

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Ctrl+S | 保存 | 仅在编辑模式下可用 |
| Ctrl+E | 编辑 | 切换到编辑模式 |
| Ctrl+F | 查找 | Monaco Editor 内置功能 |
| Ctrl+H | 替换 | Monaco Editor 内置功能 |
| Esc | 关闭 | 关闭编辑器窗口 |

## 技术细节

### 文件结构
```
FileManager/
├── components/
│   ├── CodeEditor.vue      # Monaco Editor 封装组件
│   └── FilePreview.vue     # 文件预览/编辑对话框
├── composables/            # 组合式函数
└── utils/                  # 工具函数
```

### 主要改进点
1. **CodeEditor.vue**：
   - 优化了编辑器配置
   - 启用了更多用户友好功能
   - 保持了简洁的代码结构

2. **FilePreview.vue**：
   - 简化了工具栏，移除冗余按钮
   - 优化了样式，解决视觉嵌套问题
   - 保留了核心功能按钮
