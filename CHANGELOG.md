#  项目更新记录 (CHANGELOG)

> ️ **重要提示**：基于普遍认知，我们将 `v1.0.0` 定为首个正式版本。在此之前（Pre-1.0），我们的软件随时可能进行破坏性的更新，请谨慎用于生产环境。

### [v0.2.0] - 2026-06-09

修复已知bug，引入全新的终端特性，配置系统，快捷键支持等。

####  新增功能 (Added)
- **结构化终端**：引入结构化终端，面向未来的终端设计，更加适配AI等特性。
- **终端模式切换**：支持结构化模式和经典模式运行时切换，数据同源。
- **命令补全**：100+ 内置命令，Tab 补全，支持命令、子命令、选项补全。
- **会话录制**：录制终端会话，导出为文本/HTML/JSON。
- **命令历史**：收藏、搜索、危险命令标记。
- **代码高亮**：支持 15+ 语言自动检测和语法高亮。
- **临时终端**：依附于当前会话的交互式终端，用于 Ctrl+R 等操作。
- **配置系统**：后端持久化存储，支持终端类型、字体大小、交互模式等配置。
- **断线重连**：自动/手动重连，提示符检测。
- **快捷键支持**：完整 Readline 键映射、TTY 控制信号、Alt/Meta 组合键。
- **多档位屏幕**：根据屏幕大小动态调整窗口初始大小。
- **版本信息**：显示软件版本号。
- **终端搜索**：快捷搜索终端的历史执行。aD

#### 优化细节 (Optimize)
- **AI助手**：改为使用新版本的终端
- **全局日志**：兼容新版本
- **批量命令**：兼容新版本

#### 移除功能 (Remove)
- **底部最近命令**：移除底部最近命令，改用更加合理的终端历史会话

####  修复BUG (Fix)
- **屏幕大小异常**：修复系统缩放导致的屏幕异常。
- **连接丢失**：修复断开连接后永久记录被删除的问题。
- **配置不生效**：修复配置文件实时写入问题。

### [v0.1.0] - 2026-06-05

首个测试版本正式发布！本版本奠定了系统的核心基础架构，并推出了一站式 SSH 管理控制台，为您提供从终端操作到智能运维的全链路体验。

####  新增功能 (Added)
- **️核心 SSH 模块**：首个测试版本上线，提供稳定且安全的远程连接通道。
- **️交互式终端**：支持多标签页 Web Terminal，提供流畅的命令行交互体验。
- **可视化文件管理**：内置轻量级文件浏览器，支持文件的上传、下载、编辑及目录树预览。
- **实时资源监控**：提供服务器 CPU、内存、磁盘 I/O 及网络状态的实时图表看板。
- **️防火墙管理**：支持可视化的安全组/iptables 规则配置，一键放行或拦截端口。
- **端口转发**：轻松配置本地/远程端口映射，安全穿透内网服务。
- **进程守护**：集成进程保活机制，确保关键后台任务在异常退出后自动重启。
- **AI 智能助手**：接入大语言模型，提供命令解释、报错排查及运维建议等辅助功能。
- **批量命令执行**：支持多台服务器并发下发指令，大幅提升集群运维效率。
- **全局日志审计**：集中收集并展示所有操作日志，方便回溯追踪与安全审查。
